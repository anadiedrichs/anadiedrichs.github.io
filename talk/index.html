frost: an R package - prediction of minimum temperature for frost forecasting in agriculture

2018-09-05
LatinR Conference
We present the package frost which aim is to survey prediction techniques of minimum temperature for agronomic frost prediction in viticulture and agriculture. The version 0.0.1 of the package frost introduces two empiric methods used by agronomic engineers and farmers: the first one is recommended by FAO (Food and Agriculture Organization of the United Nations), and the second one is used in Mendoza, Argentina. This package can be downloaded from its GitHub repository under the MIT open source license.
